How to install NetherOS Packages

Last updated: April 5 2026


Requirements

OS: Windows 10


Tutorial

  1. Go to the products page and select product you want to install.
  2. Select the platform you use (64-bit, 32-bit or arm64).
    • Click download portable to download portable version.
    • Click download install to download installer version.
